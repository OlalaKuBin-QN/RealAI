## 🔍 Overview

This repository provides the official implementation for **generalizable AI-generated image detection (AIGD)** using our newly proposed **RealAI dataset**, along with tools for evaluation, training, and fine-tuning.

The **RealAI dataset will be released publicly in a future update**.  
More details and download links will be provided once the release is finalized.

---

## 🗂️ RealAI Dataset

**RealAI** is a realistic and diverse AI-generated image (AIGI) dataset designed to reflect real-world detection challenges.  
It improves upon previous datasets through:

- **High realism** in generated fake images  
- **Coverage across GAN, Diffusion, and Autoregressive (AR) models**  
- **Balanced real vs. fake samples**  
- **Wide semantic diversity**  
- **Challenging evaluation of generalization to unseen generators**

📌 **Release Status:**  
The dataset is currently being prepared for public release.  
A download link and documentation will be added soon.
